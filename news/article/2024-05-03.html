New Website

2024-5-3

I've been working on the new external unit API section of the website, and after talking to a friend of mine who is a UX designer, it became obvious that I needed to redo the site.

The original site was all pure HTML, CSS and JS, but after some convincing from a different friend who is a web developer, the whole thing is now written using React and NextJS. I've also redone the home page to better display what the program is about and made sure that the site looks good on both desktop and mobile.

There are still changes to be made - and I've yet to add the aforementioned developer section - but I think this is a great upgrade and I'm thankful to my friends who pushed me into doing it who helped me see that a better design was possible.
