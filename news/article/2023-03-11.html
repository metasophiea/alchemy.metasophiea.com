Alpha 0.0.4

2023-3-11

This version adds a number of units to the Alpha collection, along with other small fixes

  • Fixed Musical Keyboard's boolean ports
  • Added the Data Readout to the Alpha collection
  • Fixed issue with the Musical Keyboard sending multiple messages when a keyboard key is held down
  • Changed base octave of the Musical Keyboard from 0 to 2
  • Added the Basic Synthesizer to the Alpha collection
  • Added the Signal Combiner to the Alpha collection
  • Added the Signal Duplicator to the Alpha collection
  • Added the Signal Readout to the Alpha collection
